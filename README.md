## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

npm, node
  ```sh
  npm install
  npm start
  npm test
  ```

### API

API routes:

- GET sentence/:id
- GET sentence/:id/translate
- POST sentence
- PUT sentence/:id
- DELETE sentence/:id

- GET sentence (This route allows orderBy, order, limit, startAt and endAt parameters)
